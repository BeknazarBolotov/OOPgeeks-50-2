Команды на один раз для устройства Windows
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"


Команды на один раз для проекта
git init


Команды которые часто пишутся в проекте
git add .       | Команда копирует и сохраняет в буфер (Невидимый)
git commit -m "commit message"      | Команды сохраняет изменения в файле
git push origin main    | Команда отправляет изменения в репозиторий и сохраняет
git branch     | Команда дял просмотра списка веток
git status     | Команда для просмотра состояния проекта
